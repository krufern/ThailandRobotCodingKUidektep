# ThailandRobotCodingKUidektep
ไฟล์ฝึกนักเรียนแข่งขันหุ่นยนต์ โครงการ Thailand Robot &amp; Coding Challenge 2026 โดยคณะวิศวกรรมศาสตร์ มหาวิทยาลัยเกษตรศาสตร์ และ iDektep พานักเรียนไปแข่งเป็นปีที่ 3 ปีนี้มีการปรับกติกาการแข่งขัน จึงได้อบรมนักเรียน 25-26-27 มีนาคม 2569 ณ ห้องประชุมเสถียรคำเนตร โรงเรียนบางบ่อวิทยาคม วิทยากรคือ พี่เจ็ท พี่แซม ขอบคุณครับผอ รองผทัยรัตน์ครับ เป้ อ.จักรกฤษ
